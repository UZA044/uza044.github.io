System Utilities Suite

This is a Java-based implementation of 3 key system utilities:

1 : File Operations: Reimplementation of basic file operations with support for piping commands, similar to Unix-based tools eg: cut , wc , cat , uniq , sort.

2 : Memory Allocator: A contiguous memory allocation system, focusing on memory management algorithms implementing a Best Fit architecture.

3 : TLB Optimization: A program that manages virtual memory by updating both the page table and the translation-lookaside buffer (TLB) as addresses are accessed. Additionally, the program implements an LRU (Least Recently Used) replacement policy to efficiently manage the TLB entries.
